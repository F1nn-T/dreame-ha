# Contributing

Contributions are welcome and appreciated! This integration aims to support the full range of Dreame and MOVA robots — mowers and vacuum cleaners alike. Community help is the only way to get there.

## How to contribute

1. Fork this repository.
2. Create a feature branch from `main`.
3. Make your changes.
4. Open a pull request against `main` with a clear description of what you changed and why.

## Adding support for a new device

If your Dreame/MOVA robot is not working or not fully supported:

- Open an issue first to describe your device model and what is missing.
- If you know what code change is needed, feel free to open a PR directly.
- Include the device model name in both the issue/PR title and description.

## Reporting bugs

- Open an issue on the [GitHub Issues page](https://github.com/F1nn-T/dreame-ha/issues).
- Include your device model, Home Assistant version, and relevant log output.
